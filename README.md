# go-mods

来源于日常开发中积累的go代码、抽出作为模块、方便以后开发使用

1. 优先尽可能引用 golang 标准库
2. 基础功能工具
   - 可引用 [Goframe](https://github.com/gogf/gf) 框架内模块化包
   - 或者 [lo](https://github.com/samber/lo) 泛型支持工具包
3. 最后是使用三方库

## preview

- **qrcode** 二维码组件
- **captcha** 验证码组件
- **idgen** ID 生成器
- **httpreq** http 请求组件
- **utils** 工具包/公共函数/便捷方法
